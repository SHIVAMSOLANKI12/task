# 🚀 Pokemon MERN App

A full-stack Pokémon application built using the MERN stack (MongoDB, Express, React, Node.js).
This app displays a list of Pokémon and allows users to view detailed information including types, moves, and game indices.

---

## 📌 Features

* 📋 Display Pokémon list
* 🔍 View Pokémon details on click
* 📑 Tab-based UI (Types, Moves, Game Indices)
* ⚡ Fast and responsive interface
* 🌐 API integration with PokeAPI
* 🧩 Clean and modular architecture

---

## 🛠️ Tech Stack

### Frontend:

* React.js
* Axios
* CSS

### Backend:

* Node.js
* Express.js
* Axios
* CORS

### API:

* PokeAPI (https://pokeapi.co/)
